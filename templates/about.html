                    SHIT was build using the following ultimate cutting-edge tech:

                      - PHP
                      - some JQuery
                      - fontawesome 
                      - Bootstrap4

                    Glftpd webui was originally based on SHIT :)
        
                    It can do user management, shows online users, glftpd status, stops/starts glftpd container
                    and can be used to view logs, edit config files and browse site. Also has a browser terminal
                    that displays tools like gl_spy, useredit and bot partyline (using websockets).

                    Additional tools and libs used:

                      - filemanager.github.io
                      - GoTTY web terminal
                      - github.io/ansi-escapes-to-html
                      - pywho
                      - pyspy (flask)
                      - gltool.sh
